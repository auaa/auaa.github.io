> npm commands

```
npm outdated                   .... checks outdated packages
npm cache clean                .... clears cache app data
npm i                          .... installs package.json
npm install -g npm@[version]   .... upgrade or downgrade npm
npm install --save             .... updates package.json dependencies
npm install --save-dev         .... updates package.json dev-dependencies
npm -v                         .... checks npm version
npm audit		       .... checks vulnerability 
```
